# 这是一个示例 Cookie 文件
# 请将你的网易云音乐 Cookie 粘贴到 cookie.txt 文件中（不是这个文件）

# 如何获取 Cookie：
# 1. 在浏览器中登录网易云音乐 (music.163.com)
# 2. 打开开发者工具 (F12)
# 3. 切换到 Network (网络) 标签
# 4. 刷新页面
# 5. 找到任意请求，查看 Request Headers 中的 Cookie
# 6. 复制完整的 Cookie 字符串
# 7. 粘贴到项目根目录的 cookie.txt 文件中

# 注意：
# - cookie.txt 文件已被添加到 .gitignore，不会被提交到代码仓库
# - 推荐使用扫码登录功能，会自动生成 login_cookie.txt
